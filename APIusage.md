###Profile
- URL: http://localhost:8080/profile
- Method: GET
- Arguments:
    ```
    {   
        (Variable  : DataType)
        "Username"   : String, #(No Special Characters)
        "Email"       : String,
        "Password"    : String, 
        "JoinedAt"    : Date, 
        "Posts"       : Object,
        "PhoneNumber" : String,
        "ProfileImg"  : String,
    }
